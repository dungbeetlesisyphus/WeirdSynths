Getting Started

From zero to face-controlled synthesis in about 10 minutes.

0 PREREQUISITES

Before you start, make sure you have these installed:

VCV Rack 2
Free or Pro edition. Download from vcvrack.com
Python 3.9+
For the face-tracking bridge. Check with: python3 --version
Webcam
Any USB or built-in camera. 720p minimum recommended.
pip (Python packages)
mediapipe, opencv-python, numpy (installed in step 2)
VOICE module only? If you just want pitch tracking, skip steps 2-3. VOICE works with any audio input — no bridge or webcam needed.

1 INSTALL THE PLUGIN

Download the latest release for your platform from GitHub and copy it into VCV Rack's plugin folder:

# macOS (Apple Silicon) cp -r WeirdSynths ~/Library/Application\ Support/Rack2/plugins-mac-arm64/ # macOS (Intel) cp -r WeirdSynths ~/Library/Application\ Support/Rack2/plugins-mac-x64/ # Linux cp -r WeirdSynths ~/.Rack2/plugins/ # Windows copy WeirdSynths %LOCALAPPDATA%\Rack2\plugins\

Restart VCV Rack completely (Cmd+Q / Alt+F4, then relaunch). You should see NERVE, SKULL, MIRROR, and VOICE in the module browser under "WeirdSynths".

Not showing up? Make sure you fully quit Rack (not just close the window). Check the Rack log at ~/.Rack2/log.txt for errors.

2 INSTALL THE PYTHON BRIDGE

The bridge runs face tracking on your webcam and sends landmark data to VCV Rack over UDP. Install the required Python packages:

pip3 install mediapipe opencv-python numpy

That's it. The bridge script itself is included with the plugin source code at bridge/nerve_bridge.py.

3 START THE BRIDGE

Open a terminal and run the bridge:

python3 bridge/nerve_bridge.py # You should see: [NERVE BRIDGE] Starting face tracker... [NERVE BRIDGE] Camera opened (1280x720) [NERVE BRIDGE] Sending to ports: [9000, 9001, 9002] [NERVE BRIDGE] Tracking active — 468 landmarks

A preview window will open showing the webcam feed with face landmark overlay. Position yourself so your face is clearly visible and well-lit.

Multiple cameras? Use --camera 1 to select a different camera index. --camera 0 is usually the default.
Custom ports? Use --port 9000 9001 9002 to specify which UDP ports to send to. Each module listens on its own port: NERVE=9000, SKULL=9001, MIRROR=9002.

4 YOUR FIRST PATCH: FACE FILTER SWEEP

The simplest and most satisfying first patch: control a filter with your eyebrow.

VCV Audio → audio in to your DAW/speakers VCO-1 (Saw wave) OUT → VCF IN NERVE L BROW → VCF CUTOFF CV (attenuate to ~50%) VCF OUT → VCA → Audio
  • Add a VCO (e.g., VCV VCO-1), set to SAW wave
  • Add a VCF (e.g., VCV VCF), patch VCO out → VCF in
  • Add NERVE from the WeirdSynths brand in the module browser
  • Patch NERVE's L BROW output → VCF's cutoff CV input
  • Attenuate the CV to ~50% so it doesn't sweep too aggressively
  • Patch VCF out → Audio module output
  • Raise your left eyebrow — the filter should open up

Once that's working, try connecting JAW to a VCA's CV input — open your mouth to let sound through. Instant expressive control.

5 FACE DRUMS WITH SKULL

For instant gratification, just add SKULL and patch its MIX output to your audio:

SKULL MIX → Audio IN That's it. Now: Blink → Kick Open jaw → Snare Raise brows → Hi-hat

Adjust SENS if triggers are too sensitive or not sensitive enough. Start around 60% and tune from there.

6 SING TO CONTROL A SYNTH WITH VOICE

VOICE doesn't need the bridge — it works with any audio input. Set up VCV Audio as an input, then:

Audio (mic input) OUT → VOICE IN VOICE V/OCT → VCO-1 V/OCT GATE → ADSR GATE ENV → VCA CV VCO-1 → VCF → VCA → Audio OUT

Hum or sing a note. The CRT display on VOICE shows the detected pitch. The VCO tracks your voice, and the gate opens when you're producing voiced sound. You're now a human oscillator controller.

Quality modes: Right-click VOICE to change quality. "Light" is fastest for speech. "Premium" gives sub-cent accuracy for musical performance but adds ~46ms latency.

7 ADD THE MIRROR

Drop MIRROR into your rack alongside NERVE. It receives face data on port 9002 automatically. Now you can see your phosphor-green wireframe face while you perform. It's part visual feedback, part aesthetic statement.

? TROUBLESHOOTING

Modules don't appear in browser
Check that the plugin folder name is exactly "WeirdSynths" (case-sensitive). Check ~/.Rack2/log.txt for ABI version errors. Ensure you fully quit Rack (Cmd+Q) before relaunching.
Bridge runs but no data in NERVE
Check that bridge says "Sending to ports: [9000, 9001, 9002]". Make sure no firewall is blocking localhost UDP. Try stopping and restarting the bridge after Rack is loaded.
Face tracking is jittery
Increase SMOOTH on NERVE to 40-60%. Ensure good lighting on your face — side-lighting and backlighting make tracking less accurate. Move closer to camera if you're far away.
VOICE not detecting pitch
Increase SENS knob. Check that audio is actually reaching the module (patch THRU to an output and listen). Try "Light" quality mode for faster detection. Ensure you're making a voiced sound (not whispering).
High CPU usage
The Python bridge is doing the heavy lifting. If CPU is high, try reducing camera resolution: --width 640 --height 480. In VOICE, try "Light" quality mode.
