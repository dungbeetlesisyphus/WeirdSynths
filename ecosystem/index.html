02 AI Backend ‚Äî Your Choice

First boot asks once. Revisit anytime in Settings. No wrong answer.

üñ• Local
Qwen2.5-Coder 1.5B runs directly on the Hailo-10H's 8GB RAM. The Pi 5 doesn't even know it's happening.
  • Fully offline ‚Äî works anywhere
  • Private ‚Äî code never leaves device
  • No API costs, no subscription
  • Zero-config, works out of box
  • ~3-8s per generation (fast enough)
  • ~1.5B param model (skill-augmented)
‚òÅÔ∏è Cloud
Routes WeirdMaker queries to Claude API. Best quality for complex DSP math, debugging, multi-file refactoring.
  • Highest response quality
  • Latest knowledge
  • Better at complex DSP/math
  • Requires internet connection
  • API key + pay-as-you-go
  • ~$0.01-0.15 per query
‚ö° Hybrid
Local for quick tasks. Escalates to cloud when it needs to. WeirdMaker decides based on task complexity.
  • Fast simple answers, powerful complex ones
  • Works offline for most tasks
  • Best cost/quality balance
  • Routing table is configurable
  • Needs API key for escalation
  • UI shows which backend was used

07 Guiding Principles
01
Offline-First
Every core feature works without internet. Network is enhancement, not requirement. WeirdBox performs equally well on a tour bus in the desert as in a studio with fiber.
02
Privacy by Default
Local AI backend is recommended at setup. Your code, your patches, your face ‚Äî none of it leaves the device unless you explicitly enable it.
03
Body as Instrument
The tracking is not a gimmick. Every module is designed around real musical expression ‚Äî what your face, hands, and voice actually do when you perform.
04
Modular Philosophy
The platform is modular. WeirdPhone works without WeirdBox. WeirdFace works standalone. NERVE works without any WeirdSynths hardware. Mix and match.
05
Open Protocol
WeirdConnect is documented and MIT-licensed. Third-party developers can build devices that speak it. The ecosystem grows beyond WeirdSynths alone.
06
Maker Culture
WeirdMaker ships with the device because the people who buy WeirdBox are also the people who want to build their own modules. User and creator are the same person.
07
Hardware as Distribution
The physical box gets WeirdSynths onto desks and into studios. Software revenue sustains the business. The box doesn't need high margin at launch.
